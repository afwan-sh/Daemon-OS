# DaemonOS

- ***DaemonOS*** is a *terminal environment* that work on java and act like a operating system handle basic task
- Like making files, folder reading and moving between them.
- It will boot up then load user data with the help of Java Serialization.
- Then ask to make user name and password if it run first time it will ask for add new user 
- For now it will only have files reading and writing not to big but small file 

### *Structure tree*
```
│
├ bootSystem
├ shell
├ terminalEmulator
└userData
```
